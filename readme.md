Exploration of the various ways one can capture and save a still image from a webcam on the Web with JavaScript. 
This includes:

* `canvas.toBlob()`
* `OffscreenCanvas`
* `createImageBitmap`
* `ImageCapture`
* `MediaStreamTrackProcessor` with `ReadableStreams`

See the post on webrtcHacks for full details: https://webrtchacks.com/still-image-from-webcam-stream-approaches/
